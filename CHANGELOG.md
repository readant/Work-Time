# Changelog

## [0.3.0] - 2026-07-16

### Added
- Activity Bar sidebar with Tree View (今日概览、快捷操作、最近会话)
- Session timer records in sidebar
- Trend chart with 7-day moving average
- Year selector for heatmap calendar

### Changed
- Sidebar UI美化
- Trend chart优化

## [0.2.0] - 2026-06-05

### Added
- Smart Pomodoro timer with adaptive focus duration recommendations
- File session timer with auto-pause/resume on editor switch
- Heatmap calendar visualization (GitHub-style contribution graph)
- Language tracking and distribution charts
- Multi-format report export (TXT, Markdown, JSON, CSV)
- Chinese localization for commands and UI
- Problem counting feature for tracking daily exercises

### Changed
- Renamed from "Coding Time" to "Work Time"
- Improved webview UI with pill-style tabs, card layout, and donut charts

### Fixed
- Multiple runtime defects in tracker initialization

## [0.1.0] - 2026-05-26

### Added
- Initial release
- Coding time tracking with Active/Idle/Away states
- Idle timeout detection
- Project and file-level statistics
- Git commit tracking
- Status bar display
- Daily stats persistence
- Basic statistics webview
